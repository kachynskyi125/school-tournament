# Školní fotbalový turnaj

## Autoři
 - Andri Kachynskyi (@kachynskyi125)

## Co projekt obsahuje
Projekt představuje jednoduchý web věnovaný školnímu fotbalovému turnaji.  
Obsahuje:
- úvodní stránku s popisem akce,
- sekci s programem a nabídkou aktivit,
- přihlašovací / kontaktní formulář,
- patičku s informacemi o autorech,
- základní stylování pomocí CSS,

## Rozdělení práce
- **Andrii Kachynskyi**:  HTML, CSS, README.  

## Postup práce
1. Návrh tématu.  
2. Vytvoření základní struktury projektu (soubory index.html, style.css).  
3. Implementace HTML obsahu – nadpis, popis, program, formulář, patička.  
4. Vytvoření a ladění CSS stylů pro jednotlivé části stránky.  
5. Testování stránky v prohlížeči a finální úpravy textů a vzhledu.

## Jak projekt spustit
Stačí stáhnout celý projekt do počítače a otevřít soubor **index.html** v libovolném webovém prohlížeči.
